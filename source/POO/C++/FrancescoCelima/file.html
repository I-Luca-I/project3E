Le Classi

Una classe è la descrizione astratta di un tipo di dato e descrive una famiglia di oggetti con caratteristiche e comportamenti simili(stessi attributi,stesse operazioni). -Nei linguaggi di programmazione, classe viene dichiarata con la parola class, e al suo interno possiamo specificare il tipo di visibilità di dati:

-public:permette l’utilizzo di variabili e metodi dall’esterno;
-private:nasconde metodi e codice che risulteranno inutilizzabili;
-protected:dati e codici nascosti ma visibili da chi li eredita. esempio in Java: 

    
        public class Rettangolo{
            private int base;
            private int altezza;
            //metodi vari
        }

        public int calcolaPerimetro(){
            return 2*(base + altezza);
        }

        public int calcolaArea(){
            return base * altezza;
        }

I costruttori inizializzano subito un oggetto al momento della creazione, esso è un metodo particolare che si distingue dagli altri per 3 elementi:
-ha lo stesso nome della classe;
-non ha tipo di ritorno;
-ha visibilità pubblica;

I metodi get/set, sono importanti per un programma in java:
-Il metodo get, restituisce il valore dell’attributo.
-il metodo set, ha lo scopo di impostare un nuovo valore all’attributo.

Esempio in c++: 

        
            double getBase(){
                return base;
            }
        
            void setBase(double valore){
                base = valore;
            }

            double getAltezza(){
                return altezza;
            }

            void setAltezza(double valore){
                altezza = valore;
            }
