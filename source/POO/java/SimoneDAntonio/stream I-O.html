Stream di I/O e l'astrazione in Java

All'interno di un programma gli stream di input/output (I/O) rappresentano un modo per leggere e scrivere dati. Un'operazione di I/O coinvolge un flusso di dati tra una sorgente (fase di input) e una destinazione (fase di output).

Stream di I/O in Java

In Java, gli stream di I/O sono rappresentati come flussi di byte. Rappresentandoli con uno schema in cui si deve inserire il proprio nome:

(Sorgente)

Nome |

Stream

Buffer

Lettura

(Destinazione)

Nome

Ci sono due stream predefiniti importanti: System.in e System.out.

System.in

System.in è uno stream di input standard predefinito in Java. Rappresenta l'input da tastiera. È possibile leggere i dati da System.in utilizzando la classe Scanner, di cui parleremo dopo. Ad esempio:

    import java.util.Scanner;

    public class Main {
      public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("Inserisci un numero intero: ");
        int numero = scanner.nextInt();
        System.out.println("Hai inserito: " + numero);
      }
    }

System.out

System.out è uno stream di output standard predefinito in Java. Rappresenta l'output su console. È possibile scrivere dati su System.out utilizzando il metodo println() o print(). Ad esempio:

    public class Main {
      public static void main(String[] args) {
        System.out.println("Questo è un messaggio di output.");
        int numero = 42;
        System.out.print("Il numero è: " + numero);
      }
    }
