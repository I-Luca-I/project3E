La classe Scanner

La classe Scanner è una classe che permette di leggere i dati di input da un flusso di byte (come la tastiera) e di convertirli in tipi di dati Java.

La classe Scanner è definita nel package java.util e per poterla utilizzare è necessario importare il package java.util.

    import java.util.Scanner;

Per utilizzare la classe Scanner è necessario passare mediante una variabile di tipo Scanner che sfrutta la libreria System.in. Infatti la classe Scanner prevede in ingresso l'oggetto di cui fare la scansione. 

    Scanner input = new Scanner(System.in);

La classe Scanner contiene al suo interno numerosi metodi per ricevere dei dati da input, ognuno diverso per ogni tipo di dato.

Funzione 'next();'

Questo metodo prende in considerazione solo il primo token inviato in input

!! Un token è la parola subito prima del delimitatore di default, in questo caso lo spazio

    import java.util.Scanner;

    public class Main {
        public static void main(String[] args) {
            Scanner scanner = new Scanner(System.in);
            System.out.println("Inserisci il tuo nome: ");
            String nome = scanner.next();
            System.out.println("Hai scritto '" + nome + "'");
        }
    }

In questo caso, inserendo un nome da tastiera e trasmettendolo in flussi di byte, verrà presa in considerazione solo la prima parola inviata, perciò inserendo più di una parola verrà inviata solo la prima.

Funzione 'nextLine();'

Questo metodo prende tutta la linea digitata prima di premere il tasto di invio

    import java.util.Scanner;

    public class Main {
        public static void main(String[] args) {
            Scanner scanner = new Scanner(System.in);
            System.out.println("Inserisci il tuo nome e il tuo cognome: ");
            String nome = scanner.nextLine();
            System.out.println("Hai scritto '" + nome + "'");
        }
    }

Funzione 'nextInt();'

Questo metodo prende in considerazione solo il primo numero inviato in input

    import java.util.Scanner;

    public class Main {
        public static void main(String[] args) {
            Scanner scanner = new Scanner(System.in);
            System.out.println("Inserisci un numero: ");
            Integer num = scanner.nextInt();
            System.out.println("Hai scritto '" + num + "'");
        }
    }

Seguendo lo stesso principio della funzione 'next();', si prende in considerazione solo il primo numero prima del delimitatore di default

Funzione 'useDelimiter();'

Questo metodo permette di cambiare il delimitatore di default, in questo caso lo spazio, con un altro delimitatore a scelta

    import java.util.Scanner;

    public class Main {
        public static void main(String[] args) {
            Scanner scanner = new Scanner(System.in);
            scanner.useDelimiter(",");
            System.out.println("Inserisci il tuo nome e il tuo cognome: ");
            String nome = scanner.next();
            System.out.println("Hai scritto '" + nome + "'");
        }
    }

Utilizzando la classe Scanner, ci si potrebbe imbattere in un'Eccezione: se in input si riceve un valore diverso dal tipo atteso questa si attiva. Non è un errore che dipende dalla sintassi del codice ma è un tipo di errore che si verifica durante l'esecuzione del programma stesso.

    import java.util.Scanner;

    public class Main {
        public static void main(String[] args) {
            Scanner scanner = new Scanner(System.in);
            System.out.println("Inserire un numero: ");
            Integer num = scanner.nextInt();
            System.out.println("Hai scritto '" + num + "'");
        }
    }

In questo caso il tipo di input atteso era un numero, essendo stata inserita una stringa il programma ha presentato l'eccezione e l'esecuzione è stata interrotta.
