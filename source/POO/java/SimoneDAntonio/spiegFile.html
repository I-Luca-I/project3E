La gestione dei File in Java

La creazione di un file di Testo

Vediamo adesso nella pratica come comportarsi con i codici scritti in precedenza. Utilizzando questo file contenente la classe di utilità possiamo iniziare.

Per prima cosa, creiamo un file di testo chiamato file.txt all'interno della cartella del progetto e inseriamo al suo interno il seguente testo:

    Hello World!

Adesso, creiamo un file Java all'interno della cartella del progetto e chiamiamolo Main.java. All'interno di questo file, inseriamo il seguente codice:

    import utils.FileUtils;

    import java.io.File;
        
    public class Test {
        public static void main (String[] args) {
            File ilMioDoc = FileUtils.creaFile("Ciao.txt");
            FileUtils.scriviFile(ilMioDoc,"Hello World!");
            FileUtils.leggiFile(ilMioDoc);
        }
    }

Se eseguiamo il codice, vedremo che verrà creato un file chiamato Ciao.txt all'interno della cartella del progetto e al suo interno verrà scritto il testo "Hello World!". Inoltre, verrà stampato a video il contenuto del file.
