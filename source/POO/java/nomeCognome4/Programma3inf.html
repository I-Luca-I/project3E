Classe String

Una stringa è formata da una lista lineare di caratteri racchiusa fra due coppie di doppi apici; n Java è gestita dalla classe String. Data l'importanza di questo tipo di dato, si possono dichiarare nuovi oggetti stringa senza usare l'operatore new. La dichiarazione può avvenire una sintassi simile alle dichiarazione dei tipi di dati primitivi String str="ciao a tutti"; System.out.println(str); o con una sintassi più formale simile alla dichiarazione dei vettori: String str=new String ("ciao a tutti");

Esistono molti metodi per la classe String e di vari tipi: Ci sono metodi di selezione come: char charAt(int i)-Ritorna il carattere con indice i. METODI DI CONFRONTO: -int length()-Restituisce la lunghezza della stringa -boolean equals(Object o)-Confronta la stringa attuale con l'oggetto specificato. METODI DI CONCATENAZIONE: String concat (String s)- concatena la stringa argomento alla stringa attuale. METODI DI CONVERSIONE: -char[] toCharArray()-Converte la stringa in un vettore di caratteri. -String toLowerCase()-Converte tutti i caratteri della stringa in minuscolo. -String toUpperCase()-Converte tutti i caratteri della stringa in maiuscolo -String trim()-Elimina gli spazi iniziali e finali. METODI DI SOSTITUZIONE: -String replace(char oldCh, char newCh)-Ritorna una nuova stringa dove tutte le occorrenze di vCh sono sostituite con nCh. -String replaceAll(String regExp,String s)-Sostituisce ogni sottostringa di questa stringa corrispondente all'espressione regolare data. -String replaceFirst(String regExp,String s)-Sostituisce la prima sottostringa di questa stringa che corrisponde all'espressione regolare data.

Metodi Statici

I metodi dichiarati nelle classi si possono distinguere in due tipologie: -metodi di istanza -metodi di classe (o static). I metodi di istanza sono utilizzabili dalle istanze della classe. Dalla classe si crea un oggetto e, utilizzando l'oggetto, si incvoca l'esecuzione dei metodi di istanza. E questi metodi possono fare riferimento agli attributi dell'oggetto, leggendo o modificando il loro valore. I metodi static sono dei metodi che possono essere utilizzati anche senza che venga creato un oggetto della classe che li contiene. Questi metodi non possono fare riferimento agli attributi definiti nella classe, a meno che non siano statici. I metodi di classe devono essere dichiarati inserendo la parola chiave static nell'intestazione del metodo. L'esecuzione di un metodo statico da altre classi si attiva indicando il nome della classe seguito dal nome del metodo.

METODO DI ISTANZA class Operazione { private int a, b; public Operazione(int a, int b) { this.a=a; this.b=b; } public int somma() { return a + b; } } Il codice definisce un metodo di istanza e due attributi: per la sua esecuzione è richiesta la creazione di una istanza e l'esecuzione del metodo d'istanza. METODO DI CLASSE class Operazione { public static int somma (int a, int b) { return a + b; } Questo codice definisce un metodo di classe con due parametri: per la sua esecuzione si usa direttamente il nome della classe seguito dal nome del metodo e dei parametri. Per eseguire la somma dei due numeri si deve usare la classe Operazione nel seguente modo: METODO DI ISTANZA int risultato; Operazione op= new Operazione (2,5); risultato = op.somma(); METODO DI CLASSE int risultato; risultato= Operazione.somma(2,5);

Quindi i vantaggi di un metodo statico, rispetto ad uno di istanza sono: -non si deve istanziare la classe per utilizzarlo; -si risparmia memoria perché non si creano oggetti; -non è necessario conoscere i dettagli della classe: il metodo usa solo i parametri e nessun attributo.
