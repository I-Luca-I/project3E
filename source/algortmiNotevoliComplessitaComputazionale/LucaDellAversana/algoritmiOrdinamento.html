Algoritmi notevoli di ordinamento

Algoritmi notevoli di ricerca >

Gli algoritmi notevoli di ordinamento servono per elencare gli elementi di un insieme di dati omogenei, secondo una relazione d'ordine di tipo crescente o decrescente, in modo che ogni elemento sia minore o maggiore del successivo;
Per il resto dell'argomento parleremo solamente di un ordinamento crescente per evitare confusione (anche se tutti gli algoritmi si possono applicare anche in ordine decrescente)

Exchange sort

L'exchange sort consiste nel confrontare ogni elemento con tutti i suoi successivi, se un elemento successivo dovesse essere minore dell'elemento corrente, questi vengono scambiati.

Esempio:

6 2 5 1

Prima iterazione:

6 2 5 1

6 > 2 =>

2 6 5 1


2 6 5 1

2 < 5

2 6 5 1

2 > 1 =>

1 6 5 2

Seconda iterazione:

1 6 5 2

6 > 5 =>

1 5 6 2


1 5 6 2

5 > 2 =>

1 2 6 5

Terza iterazione:

1 2 6 5

6 > 5 =>

1 2 5 6


1 2 5 6

Esempio di una implementazione dell'exchange sort in c++:

void exchangeSort(int a[], int n) {
  for (int i=0; i<n-1; i++) {
    for (int j=i+1; j<n; j++) {
      if (a[i] > a[j]) {
        int holder = a[i];
        a[i] = a[j];
        a[j] = holder;
      }
    }
  }
}

L'exchange sort ha una complessità computazionale di O(n2) (o quadratica)

Selection sort

Il selection sort consiste nel separare la sequenza in due sottosequenze: una con tutti i valori ordinati, l'altra con i valori da ordinare.
Si seleziona il valore più piccolo nella sottosequenza da ordinare e lo si sposta nell'ultima posizione della sequenza ordinata, si ripete così finché l'insieme non è riordinato.

Esempio di una implementazione del selection sort in c++:

void selectionSort(int a[], int n) {
  for (int i=0; i<n-1; i++) {
    int posizioneMinimo = i;	
    for (int j=i+1; j<n; j++) 
      if (a[posizioneMinimo] > a[j]) posizioneMinimo = j;	
    if (posizioneMinimo != i) {
      int holder = a[i];
      a[i] = a[posizioneMinimo];
      a[posizioneMinimo] = holder;
    }
  }
}

Il selection sort ha una complessità computazionale di O(n2) (o quadratica)

Bubble sort

Il bubble sort consiste nello scambio di elementi adiacenti se il loro ordine dovesse essere sbagliato, il procedimento continua finché non siamo sicuri che l'intera lista sia ordinata.
Il nome deriva da come gli elementi più grandi "galleggiano" verso la fine della lista, come le bolle in un liquido.

Per migliorare l'efficienza del bubble sort si possono applicare le seguenti modifiche all'algoritmo:

  • Sentinella: la sentinella è una variabile booleana che tiene traccia se è stato eseguito uno scambio in una iterazione; se per la fine dell'iterazione non è avvenuto alcuno scambio allora significa che la lista è già ordinata e possiamo smettere.
  • Variabile stop: alla fine di ogni iterazione sappiamo che almeno un elemento è stato ordinato (partendo dal più grande), possiamo quindi evitare di ricontrollare questi elementi ogni singola iterazione utilizando una variabile che separa l'insieme in due sottoinsiemi: uno da ordinare e uno ordinato.

Esempio di bubble sort con sentinella e variabile stop:

6 2 5 1

Prima iterazione:

6 2 5 1

6 > 2 =>

2 6 5 1


2 6 5 1

6 > 5 =>

2 5 6 1


2 5 6 1

6 > 1 =>

2 5 1 6


2 5 1 6

Seconda iterazione

2 5 1 6

2 < 5



2 5 1 6

5 > 1 =>

2 1 5 6

Terza iterazione

2 > 1 =>

1 2 5 6

Esempio di una implementazione del bubble sort in c++:

void bubbleSort(int a[], int n) {
  for (int i=0; i<n-1; i++) {
    for (int j=0; j<n-1; j++) {
      if (a[j] > a[j+1]) {
        int holder = a[j];
        a[j] = a[j+1];
        a[j+1] = holder;
      }
    }
  }
}

Esempio di una implementazione del bubble sort con sentinella in c++:

void bubbleSortSentinella(int a[], int n) {
  bool scambio;
  do {
    scambio = false;
    for (int j=0; j<n-1; j++) {
      if (a[j] > a[j+1]) {
        int holder = a[j];
        a[j] = a[j+1];
        a[j+1] = holder;
        scambio = true;
      }
    }
  } while (scambio);
}

Esempio di una implementazione del bubble sort con sentinella e variabile stop in c++:

void bubbleSortSentinellaStop(int a[], int n) {
  int stop = n-1;
  bool scambio;
  do {
    scambio = false;
    for (int j=0; j<n-1; j++) {
      if (a[j] > a[j+1]) {
        int holder = a[j];
        a[j] = a[j+1];
        a[j+1] = holder;
        scambio = true;
      }
    }
    stop--;
  } while (scambio);
}

Il bubble sort ha una complessità computazionale di O(n2) (o quadratica)
Le modifiche aggiunte al bubble sort diminuiscono la complessità computazionale solamente nel caso migliore, che diventa O(n) (o lineare); il caso peggiore rimane O(n2) (o quadratica)

Quick sort

Il quick sort utiliza la ricorsione e sfrutta la tecnica divide et impera.
Il quick sort consiste nella scelta di un pivot (che di solito è l'elemento centrale) e lo si confronta con tutti gli elementi nell'insieme; tutti gli elementi più piccoli vengono posizionati alla sua sinistra e quelli più grandi alla sua destra; poi si riapplica il procedimento ai due sottoinsiemi.

Esempio:

6 8 1 5 2 3 9 7

Prima iterazione:

6 8 1 5 2 3 9 7


2 3 1 5 6 8 9 7

Seconda iterazione:

2 3 1 5 6 8 9 7


1 2 3 5 6 7 8 9


1 2 3 5 6 7 8 9

Questo esempio tratta di un caso abbastanza favorevole per questo algoritmo poiché il primo pivot è casualmente circa il valore medio dell'insieme, l'algoritmo sarebbe meno efficiente se questo non fosse stato il caso.

Esempio di una implementazione del quick sort in c++:

void quickSort(int a[], int n, int inizio, int fine) {
  int pivot;
  if (inizio < fine) {
    pivot = inizio;
    int i = inizio;
    int j = fine;
    while (i < j) {
      while (a[i] <= a[pivot] and i < fine) i++;
      while (a[j] > a[pivot]) j--;
      if (i < j) {
        int holder = a[i];
        a[i] = a[j];
        a[j] = holder;
      }
    }
    int holder = a[pivot];
    a[pivot] = a[j];
    a[j] = holder;
		
    quickSort(a, (j-inizio), inizio, j-1);
    quickSort(a, (fine-j), j+1, fine);
  }
}

Nel caso migliore il quick sort ha una complessità computazionale di O(n log2 n) (o linearitmica)
Nel caso peggiore ha invece una complessità computazionale di O(n2) (o quadratica)

Merge sort

Il merge sort utiliza la ricorsione e sfrutta la tecnica divide et impera.
Il merge sort è suddiviso in due parti:
1. divide: dove l'insieme viene suddiviso in tanti sottoinsiemi ognuno di grandezza 1 (ognuno dei sottoinsiemi sono già ordinati poiché un insieme di grandezza 1 può avere solo un ordine).
2. impera: dove gli insiemi vengono uniti (da cui deriva il nome merge) tra di loro a coppie di due e ordinati. Per fare ciò si estrae ripetutamente il minimo delle due sottosequenze e lo si pone nella sequenza in uscita, che risulterà ordinata.

Esempio di una implementazione del merge sort in c++:

void mergeSort(int a[], int inizio, int fine) {
  if (inizio < fine) {
    int centro = (inizio + fine) / 2;
    mergeSort(a, inizio, centro);
    mergeSort(a, centro+1, fine);
    merge(a, inizio, centro, fine);
  }
}

void merge(int a[], int inizio, int centro, int fine) {
  int i = inizio;
  int j = centro + 1;
  int k = 0;
  int b[fine - inizio + 1];

  while (i <= centro and j <= fine) {
    if (a[i] < a[j]) {
      b[k] = a[i];
      i++;
    } else {
      b[k] = a[j];
      j++;
    }
    k++;
  }

  while (i <= centro) {
    b[k] = a[i];
    i++;
    k++;
  }

  while (j <= fine) {
    b[k] = a[j];
    j++;
    k++;
  }

  for (k=inizio; k<=fine; k++) {
    a[k] = b[k-inizio];
  }
}

Il merge sort ha una complessità computazionale di O(n log n) (o linearitmica)
