Algoritmi notevoli di ricerca

>

Gli algoritmi notevoli di ricerca servono per trovare un valore all'interno di una lista ordinata o meno di valori.

Ricerca completa

Se il vettore non è ordinato, l'unico algoritmo che possiamo utilizare è la ricerca completa.

Nella ricerca completa si controlla sequenzialmente tutti gli elementi del vettore fino a quando non si trova il valore ricercato, oppure finché non si raggiunge la fine del vettore.

Esempio di una implementazione di una funzione di ricerca completa in c++:

int ricercaCompleta(int a[], int n, int k) {
        for (int i=0; i<n; i++) {
            if (a[i] == k) return i;
        }
        return -1;
}

La ricerca completa ha una complessità computazionale di O(n) (o lineare).

Ricerca binaria (o dicotomica)

Se il vettore è ordinato in qualsiasi modo (dal più grande al più piccolo, dal più piccolo al più grande, dalla a alla z ecc.) allora è possibile utilizare la più efficiente ricerca binaria.

Nella ricerca binaria si selezione l'elemento centrale m, dividendo così l'insieme in due sottoinsiemi A (con valori minori di m) e B (con valori maggiori di m), se l'elemento cercato k è uguale a m l'algoritmo termina con successo, altrimenti se m è maggiore di k si riesegue l'algoritmo al sottoinsieme A altrimenti si riesegue al sottoinsieme B.

Esempio di una implementazione di una funzione di ricerca binaria in c++:

int ricercaBinaria(int a[], int n, int k) {
    int sx = 0, dx = n;
    
    while (sx <= dx) {
        int m = (sx + dx) / 2;
        if (k > a[m]) sx = m+1;
        else if (k < a[m]) dx = m-1;
        else return m;
    }

    return -1;
}

La ricerca binaria ha una complessità computazionale di O(log n) (o logaritmica).
